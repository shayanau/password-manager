# 🔐 Password Manager (Python)

A simple and secure password manager built with Python.

## 🚀 Features
- Add passwords
- View saved passwords
- Delete passwords
- Search passwords
- Generate strong passwords
- Encryption using Fernet

## 🛠️ Technologies
- Python
- JSON
- Cryptography (Fernet)

## ▶️ How to Run

```bash
pip install cryptography
python main.py
